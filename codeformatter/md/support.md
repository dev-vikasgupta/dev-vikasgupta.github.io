## Frequently Asked Questions

### Does CodeFormatter send my code to external servers?

No.

CodeFormatter processes formatting directly within Google Workspace and does not send document content to external servers.

---

### Why does CodeFormatter require access to the current document?

The add-on must read the selected code and write the formatted result back into the document.

CodeFormatter only requests access to the currently active document.

---

### Can I undo formatting?

Yes.

Google Docs supports standard undo functionality using:

* Ctrl + Z (Windows/Linux)
* Cmd + Z (Mac)

---

### Which languages are supported?

CodeFormatter currently supports multiple popular programming languages including JavaScript, TypeScript, Python, Java, HTML, CSS, JSON, and more.
