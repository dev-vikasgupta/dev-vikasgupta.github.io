Format and syntax-highlight code blocks directly inside Google Docs.

CodeFormatter helps developers, educators, and technical writers create clean, readable code snippets without leaving Google Docs.

---

## Features

### Syntax Highlighting

Apply syntax highlighting to code snippets in multiple programming languages, making documentation easier to read and review.

### Multiple Languages

Supports popular languages including:

* JavaScript
* TypeScript
* Python
* Java
* HTML
* CSS
* JSON

Additional languages may be added in future releases.

### Multiple Themes

Choose from light and dark themes to match your preferred documentation style.

### Works Inside Google Docs

Format code directly within Google Docs using a simple sidebar workflow.

No copy-pasting into external tools required.

---

## How It Works

1. Select your code in Google Docs.
2. Open the CodeFormatter sidebar.
3. Choose a language and theme.
4. Click **Format**.
5. Your code is instantly transformed into a readable, syntax-highlighted block.

---

## Privacy First

CodeFormatter is designed with privacy in mind.

* No document content is stored.
* No code is sent to external servers.
* Processing occurs entirely within Google Workspace.
* Only the currently active document is accessed.

For details, see the [Privacy Policy](../privacy-policy.html).

## Frequently Asked Questions

### Does CodeFormatter send my code to external servers?

No.

CodeFormatter processes formatting directly within Google Workspace and does not transmit document content to third-party services.

### Can I undo formatting?

Yes.

Google Docs standard undo functionality is supported.

### Which languages are supported?

CodeFormatter currently supports multiple popular programming languages including JavaScript, TypeScript, Python, Java, HTML, CSS, and JSON.

### Does CodeFormatter require a paid subscription?

The current release is available without a subscription.

Future premium features may be introduced separately.
